## API Rest serverless para recursos do site

    <p>Esta é uma aplicação muito pequena que utiliza express como framework node<p>
        <h3>Para que serve?</h3>
        <p>Serve para os recursos de contacto, e subscrição da newsletter</p>
    <h4>Como rodar</h4>
    <p>Na raíz do projecto execute <code>npm install</code> ou <code>yarn install</code> para instalar as dependências.</p>
    <p>Em seguida execute <code>npm start</code> ou <code>yarn start</code> para rodar a versão para produção</p>
